## Commands
prefix: **#**
| Command | Description |
| ------ | ------ |
| play | play a youtube video |
| leave | leave channel |
| loop | repeat playlist or song |
| nowplaying | Display actual music |
| pause | set stream on pause |
| queue | display songs |
| resume | set stream on resume |
| skip | skip music |
| skipto | skipc music |
| volume | change volume |
| remove | remove music from playlist |
| seek | set seek |
